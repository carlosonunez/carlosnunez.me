# carlosnunez.me

This is my web site.

## Running it locally

1. Fetch Carlos's Blog Generator:

   ```sh
   git clone https://github.com/carlosonunez/https-hugo-bloggen \
    ./blog-gen
   ```

2. Start it up locally:

    ```sh
   ./blog-gen/scripts/deploy.sh --test
   ```

   The site will be running at https://localhost:8080.
